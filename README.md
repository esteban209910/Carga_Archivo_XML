# Carga_Archivo_XML
📊 Carga de Archivos XML a SQL Server
📌 Descripción

Este proyecto tiene como objetivo cargar archivos en formato XML a una base de datos en SQL Server. Es un ejercicio enfocado en el manejo de datos semiestructurados y su transformación para almacenamiento en bases de datos relacionales.

⚙️ Tecnologías utilizadas
SQL Server
XML
T-SQL
🚀 Objetivo

Procesar archivos XML y almacenarlos en tablas de SQL Server para su posterior análisis.
