# Carga de Archivos XML a SQL Server

##  Descripción
Este proyecto tiene como objetivo cargar archivos en formato XML a una base de datos en SQL Server.  
Es un ejercicio enfocado en el manejo de datos semiestructurados y su transformación para almacenamiento en bases de datos relacionales.

---

##  Tecnologías utilizadas
- SQL Server  
- XML  
- T-SQL  

---

## Objetivo
Procesar archivos XML y almacenarlos en tablas de SQL Server para su posterior análisis.

---
 

##  Proceso general
1. Lectura del archivo XML  
2. Parseo de la información  
3. Inserción de datos en tablas SQL Server  

---

## 📈 Resultados esperados
- Datos XML correctamente almacenados en tablas relacionales  
- Estructura lista para consultas y análisis
